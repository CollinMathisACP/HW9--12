Zen Garden

These example webpages display the vast potential CSS has to compliment the content of a webpage, or even change the vibe entirely. All of these webpages contain the same content, so the layout, color schemes, etc. are responsible for how each one is perceived by the reader. One thing I find interesting is just how much the CSS is responsible for the tone of the page. The presentation is the first thing that anyone notices, so it is up to the front end developer to ensure that the page meshes well with the its content.
